# FittedForYou — Privacy Policy

This repository hosts the official Privacy Policy for the FittedForYou mobile application.

## About FittedForYou

FittedForYou is a cross-platform mobile app that suggests daily outfits based on your 
local weather forecast and personal wardrobe. Available on iOS and Android.

## Privacy Policy

The privacy policy for FittedForYou is available at:
https://yourusername.github.io/fittedforyou-privacy

## Data We Collect

FittedForYou collects the following data to provide its service:

- **Email address** — used for account creation and login
- **Location data** — used to fetch local weather for outfit suggestions
- **Outfit history** — logged by the user to improve personalized suggestions
